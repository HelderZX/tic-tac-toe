## 🔹 About

A simple Tic Tac Toe game built to practice JavaScript, programming logic, and Local Storage.

## ✨ Features

- Player vs Player mode
- Keeps game state in Local Storage
- Detects wins and draws
- Reset functionality

## 📝 How to Play

1. Open `index.html` in your browser.
2. Click on a cell to make your move.
3. The game alternates turns automatically.
4. The winner is displayed when three marks align horizontally, vertically, or diagonally.
5. Use the reset button to start a new game.

## 💻 Tech Stack

- HTML, CSS, JavaScript
- Local Storage for saving game state

## 🎯 Learning Goals

- Practice JavaScript DOM manipulation
- Implement game logic
- Work with Local Storage
